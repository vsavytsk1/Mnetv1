# Reference Sources
## Minimal Math Primitives Research

### REF 1: Graph Rewriting — Formal Foundation
**Rozenberg, G. (1997). Handbook of Graph Grammars and Computing by Graph Transformations, Vol 1-3**
- Graph rewriting: L -> R with interface K
- DPO (double pushout): pattern match, replace, preserve context
- SPO (single pushout): simpler, allows dangling edges
- Key insight: ALL computation is subgraph matching + replacement
- URL: https://en.wikipedia.org/wiki/Graph_rewriting

### REF 2: Wolfram Physics Project — Everything is a Graph
**Wolfram, S. (2020). A Project to Find the Fundamental Theory of Physics**
- Hypergraph rewriting as foundation of physics
- Space = graph structure. Time = graph updates. 
- Quantum mechanics = multiway graph (branching rewrites)
- Rule 110: simplest Turing-complete graph rewrite (proved by Cook 2004)
- Controversial but structurally sound: if universe is computable, it's graph rewrite
- URL: https://www.wolframphysics.org/

### REF 3: Category Theory — Mathematics of Mathematics  
**Mac Lane, S. (1971). Categories for the Working Mathematician**
**Lawvere, F.W. & Schanuel, S. (2009). Conceptual Mathematics**
- Objects + Morphisms + Composition = minimal math framework
- Functors = maps between categories (preserve structure)
- Natural transformations = maps between functors
- Yoneda lemma: objects ARE their relationships (nodes = their edges)
- Adjunctions: the deepest pattern (free/forgetful, curry/uncurry)
- Key insight: identity + composition + associativity = enough
