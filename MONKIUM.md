### Tool 3: SHOW DONT EXPLAIN (First 90 Seconds Rule)
`
WHY IT WORKS:
  Explanation = monkey must compute new axioms.
  Computation = resources used = threat-check opportunity.
  SHOWING = monkey sees it work = instant belief.
  No computation needed. No resources spent.
  
HOW TO USE IT:
  Show the magic working. Do not name it.
  Show the consequence. Do not explain it.
  Show the character acting. Do not describe them.
  Monkey: "I see it. I believe it. I am IN."
  
  If you explain first: monkey says "prove it."
  If you show first: monkey says "I already believe."
  
THE 90 SECOND RULE:
  World must feel real within 90 seconds.
  After 90 seconds: monkey either IN or OUT.
  No second chances on first impression.
  This is not arbitrary. This is neuroscience.
  
COST: discipline.
RETURN: monkey locked in forever.
`

`

HOW TO USE IT:

  FMA: terror -> baby Ed laughing -> terror.

  The valley is not wasted time.
  The valley IS the teaching.
  Monkey consolidates during valleys.
  
COST: courage to put in the quiet moments.
RETURN: monkey stays for the full journey.
