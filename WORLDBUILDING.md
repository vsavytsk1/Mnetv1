`
1. Motion/sound before explanation
   Monkey: "I see it work -> I believe it -> locked"

2. Self-similarity
   Show rule at small scale.
   Monkey extrapolates to large scale automatically.
   No re-explanation needed.

3. Human scale anchor
   Drop in a small human detail.
   Monkey: "oh yes this is reality now"
   Stops computing. Enjoys. Rests.
   THEN hit with the big idea while monkey is relaxed.

4. Symmetry = safe container
   Dangerous ideas become thinkable
   when presented inside symmetric geometry.
   The circle CONTAINS the fractality.
   Monkey brain: less compute. more safe. explore freely.

5. Tension-release cycle
   Spike fear/awe -> release with human moment -> repeat
   Monkey brain rhythm. Heartbeat pattern.
   Cannot sustain maximum terror. Needs valleys.
`
